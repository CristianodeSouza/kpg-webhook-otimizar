# KPG IMAGEM METADADOS

**Projeto de Otimização de Metadados em Imagens para SEO e Google Meu Negócio**

---

## 📋 Objetivo

Gerenciar e otimizar metadados de imagens PNG/JPEG de produtos imobiliários da KPG Imóveis para:
- ✅ Melhor indexação no Google Images
- ✅ Otimização SEO para buscas geográficas
- ✅ Publicação em Google Meu Negócio com metadados completos
- ✅ Melhor ranqueamento em pesquisas por "imóveis Gramado", "imóveis Canela", etc.

---

---

## 🚀 Fluxo Resumido

1. **Usuário fornece imagem** (PNG ou JPEG)
2. **Eu edito metadados** no arquivo original (sobrescrevendo)
3. **Eu verifico metadados** com script de validação
4. **Se OK** → Aviso: "✅ Imagem pronta para postar no Google Meu Negócio"

---

## 📊 Metadados Otimizados para SEO

O projeto mantém um template com **65+ metadados** otimizados:

- **Básicos**: Title, Description, Author, Copyright
- **Geolocalização**: Latitude, Longitude, GPS, City, State, Country
- **SEO**: Keywords, Subject, Category, SearchEngineOptimization
- **Contato**: Website, Email, Contact, Organization
- **Estruturados**: SchemaOrg (JSON-LD para Google)
- **Qualidade**: ImageWidth, ImageHeight, Resolution, ColorMode
